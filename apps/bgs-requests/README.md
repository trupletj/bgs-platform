# bgs-requests

BGS superapp-ийн "Хүсэлтүүд" mini-app (Next.js 16). Ажилтны төрөл бүрийн хүсэлтийг нэг дороос удирдана.

## Хөгжүүлэлт

```bash
pnpm --filter bgs-requests dev   # localhost:3002
```

`.env.local` үүсгэхдээ `.env.example`-ийг хуулна.

## Embedding

Mobile superapp / bgs.mn-ээс iframe (web) / WebView (native)-ээр embed хийгдэнэ. Token bridge болон auto-resize-ийн дэлгэрэнгүйг `CLAUDE.md`-ээс уншина уу.
